Mobile Menu — No Keyboard Dismiss Warning
The mobile overlay backdrop closes on click but lacks keyboard support for accessibility. Escape key works (custom handler), but the backdrop itself is not focusable.
src/components/Navbar.tsx:105
Add role="button" and onKeyDown to backdrop
No Error Logging Warning
Query errors are caught silently with .catch(() => []). No logging to console or external service, making debugging production issues difficult.
src/app/page.tsx, sitemap.ts
Add console.error or external logger in catch blocks
Unused Components Warning

ScrollReveal.tsx, BlogSidebar.tsx
Wire ScrollReveal into homepage sections, or remove if not needed
